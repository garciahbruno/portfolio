# Hold Images

Place your real transparent cutout hold images in this folder.

Required naming:
- `hold-01.png`
- `hold-02.png`
- ...
- `hold-40.png`

Guidelines:
- PNG with transparent background only
- No baked background box
- Prefer square canvas around 800x800+

Sources:
- Rock City product cutouts
- Cheetah Holds product cutouts

If you want fewer files, edit `/hold-images.js` and keep only the files you provide.
